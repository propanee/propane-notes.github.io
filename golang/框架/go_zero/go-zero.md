# Go-zero

github地址：[go-zero github](https://github.com/zeromicro/go-zero)   

文档地址：[go-zero docs](https://go-zero.dev/docs/concepts/overview)

微服务框架go-zero是一个集成了各种工程实践的 web 和 rpc 框架。通过弹性设计保障了大并发服务端的稳定性，经受了充分的实战检验。
